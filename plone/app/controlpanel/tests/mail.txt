Mail control panel
==================

Test setup::

    >>> app = layer['app']
    >>> from plone.testing.z2 import Browser
    >>> browser = Browser(app)
    >>> browser.handleErrors = False
    >>> browser.addHeader('Authorization', 'Basic admin:secret')
    >>> portal = layer['portal']
    >>> portal_url = portal.absolute_url()

First some initial setup code:

    >>> from zope.component import getUtility
    >>> from plone.registry.interfaces import IRegistry
    >>> from plone.app.controlpanel.interfaces import IMailSchema
    >>> registry = getUtility(IRegistry)
    >>> settings = registry.forInterface(IMailSchema)


Viewing the mail control panel
--------------------------------

    >>> browser.open(portal_url + '/@@mail-controlpanel')
    >>> browser.url.endswith('mail-controlpanel')
    True

Click the cancel button:

    >>> browser.getControl(name="form.buttons.cancel").click()
    >>> browser.url.endswith('plone_control_panel')
    True

There should be still no changes:

    >>> 'Changes canceled.' in browser.contents
    True

Make some changes
-----------------

    >>> browser.open('http://nohost/plone/@@mail-controlpanel')
    >>> browser.url.endswith('mail-controlpanel')
    True

    >>> browser.getControl(name='form.widgets.smtp_host').value = 'localhost2'
    >>> browser.getControl(name='form.widgets.smtp_port').value = '2525'
    >>> browser.getControl(name='form.widgets.smtp_userid').value = 'admin'
    >>> browser.getControl(name='form.widgets.smtp_pass').value = 'secret'
    >>> browser.getControl(name='form.widgets.email_from_name').value = 'Spambot'
    >>> browser.getControl(name='form.widgets.email_from_address').value = 'spam@localhost'

Click the save button:

    >>> browser.getControl(name="form.buttons.save").click()
    >>> browser.url.endswith('mail-controlpanel')
    True

We should be informed that something has changed:

    >>> 'Changes saved.' in browser.contents
    True

Make sure the changes have been applied correctly to the settings:

    >>> settings.smtp_host
    u'localhost2'

    >>> settings.smtp_port
    2525

    >>> settings.smtp_userid
    u'admin'

    >>> settings.smtp_pass
    u'secret'

    >>> settings.email_from_name
    u'Spambot'

    >>> settings.email_from_address
    'spam@localhost'
